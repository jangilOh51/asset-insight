## 2. 연령대별 UX 선호 분석

### 2-1. 20대 (Gen Z, 1997~2006년생)

> **키워드:** 즉각성 · 투명성 · 게임화 · 공유욕구

**투자 행태**
- 소액 분산투자 선호, ETF/테마주 중심
- 앱 평점/리뷰 후 다운로드 결정 (구전 의존)
- 손실 허용도: 중간 (잃어도 경험으로 인식)
- 정보 소비: 유튜브 > 커뮤니티 > 증권사 리포트

**선호 UX 특성**
```
✅  다크모드 기본값
✅  풍부한 모션/애니메이션 (수익 시 confetti)
✅  큰 숫자, 큰 임팩트 (정보보다 감정)
✅  바텀 탭 3~4개 (단순 내비게이션)
✅  SNS 공유 카드 자동 생성
✅  수수료/수익 투명 공개
✅  마이크로카피 친근한 말투 ("잘하고 있어요! 🎉")
❌  복잡한 표/그리드
❌  은행 느낌 네이비+골드
❌  3단계 이상 메뉴 depth
```

**컬러 반응**
| 색상 | 반응 |
|------|------|
| 보라(#7C3AED) | 개성, MZ감성, 선호 |
| 네온그린 | 게임/테크 느낌, 호감 |
| 전통 네이비 | 구식, 기성세대 느낌 |
| 파스텔 | 귀엽지만 진지하지 않음 |

### 2-2. 30대 (Millennials, 1981~1996년생)

> **키워드:** 정보균형 · 자율성 · 장기계획 · 효율

**투자 행태**
- 결혼/내집마련/자녀 등 목돈 필요 → 중기 목표 투자
- 주식+ETF+연금 복합 포트폴리오
- 모바일+데스크탑 병행 사용
- 세금/절세 정보에 민감 (ISA, IRP 활용)

**선호 UX 특성**
```
✅  중간 정보 밀도 (핵심만 보여주되 자세히보기 제공)
✅  수익률 트렌드 차트 (일/주/월/연)
✅  벤치마크 비교 (내 수익률 vs KOSPI)
✅  자동화 기능 (자동매수, 리밸런싱 알림)
✅  절세 캘린더/손익 시뮬레이터
✅  다크/라이트 선택 가능
✅  목표 진행률 시각화
❌  너무 단순한 UI (정보 부족 불안)
❌  과도한 애니메이션 (집중 방해)
```

**컬러 반응**
| 색상 | 반응 |
|------|------|
| 사이언(#06B6D4) | 데이터/분석 느낌, 신뢰 |
| 딥블루(#1E40AF) | 안정감, 선호 |
| 전체 어두운 테마 | OK, 다크모드 선호 50% |

### 2-3. 40대 (Gen X, 1965~1980년생)

> **키워드:** 신뢰 · 전문성 · 통합조망 · 안정성

**투자 행태**
- 금융 자산 규모 최대 세대
- 창구 → HTS → MTS 전환 경험
- 장기 자산관리, 연금 관리, 상속 플래닝
- 수익보다 리스크 관리 중시
- 전문가 의견/리포트 신뢰

**선호 UX 특성**
```
✅  높은 가독성 (큰 글씨, 높은 명도 대비)
✅  전체 자산 한눈에 (주식+예금+연금 통합)
✅  보안 인증 시각적 강조 (신뢰 배지)
✅  전문가 리포트/AI 분석 내장
✅  데스크탑 HTS 수준 정보량
✅  월간/연간 요약 리포트
✅  세금 시뮬레이터
❌  과도한 모션/팝업
❌  작은 탭 영역 (미스터치 우려)
❌  모호한 투자 용어 (명확한 설명 필요)
```

**컬러 반응**
| 색상 | 반응 |

| 골드(#C9A84C) | 자산/신뢰/프리미엄 |

| 밝은 흰색 배경 | 눈 피로 (다크 선호) |

```

₩2,000,000,000 →  ₩20억           (1억 단위 이상)

$1,471.97/USD  →  ₩1,472 로 병기
수량           →  1,817주 (콤마 구분)
```

### 6-3. 로딩/상태 처리

```
로딩 중:    스켈레톤 UI (공백 = 오류 인식 방지)

실시간:     마지막 업데이트 HH:MM:SS 항상 표시
오프라인:   "오프라인 — 마지막 데이터: 18:00" 배너
```

### 6-4. 접근성

```
최소 폰트:  UI 11px, 본문 13px 이상
색맹 대응:  색상 외 아이콘 병기 (▲ 상승 / ▼ 하락)

```
