# screen-09 AI 월간 리포트 (`/report`)

> last_updated: 2026-04-20  
> version: 1.0  
> sprint: S3-01

---

## 화면 목적

보유 포트폴리오를 기반으로 Claude AI가 생성한 월간 투자 리포트를 확인한다.
리포트는 월간 성과 분석, 포지션 분석, 리밸런싱 제안, 리스크 항목을 포함한다.

---

## UI 상태 (5가지)

| 상태 | 트리거 | 표시 |
|------|--------|------|
| 기본 | 페이지 최초 진입 | 연월 선택기 + "리포트 생성" 버튼 |
| 로딩 | 버튼 클릭 후 API 응답 대기 | 스피너 + "AI가 리포트를 작성 중입니다..." |
| 성공 | API 응답 수신 | 리포트 전문 표시 (섹션별 카드) |
| 에러 | API 오류 | 에러 메시지 + 재시도 버튼 |
| 빈 화면 | ANTHROPIC_API_KEY 미설정 | 설정 안내 메시지 |

---

## 레이아웃

```
AppLayout (title="AI 리포트")
└── max-w-3xl mx-auto
    ├── 헤더: 연월 선택기 (year/month select) + 생성 버튼
    ├── 캐시 안내: "캐시된 리포트 / 새로 생성됨" 뱃지
    └── 리포트 영역
        ├── [로딩] 스피너 + 텍스트
        ├── [에러] 에러 카드 + 재시도 버튼
        └── [성공] 리포트 섹션 카드들
            ├── 월간 성과 요약
            ├── 포지션 분석
            ├── 리밸런싱 제안
            ├── 주요 리스크
            └── 다음 달 관심 사항
```
