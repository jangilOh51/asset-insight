# 계좌 관리 — 화면 요구사항

## 목적
증권사 API 계좌를 등록·수정·삭제·검증하고 활성/비활성을 관리한다.
환경변수로 등록된 계좌(env-kis)도 목록에 포함하여 표시한다.

---

## 기능 요구사항

### FR-01 계좌 목록
- 계좌 카드: 브로커 아바타 / 계좌번호 / 상태 배지 / 모의 배지 / 계좌 별칭·API 연결 여부
- 상태 배지: 활성+검증됨(cyan) / 활성+미검증(amber) / 비활성(gray)
- 계좌 없으면 빈 상태 아이콘 + "등록된 계좌가 없습니다"

### FR-02 통계 카드
- 전체 계좌 수 / 활성 수 / 검증됨 수를 3열 카드로 표시

### FR-03 계좌 추가 모달
- 증권사 선택 버튼: KIS / 키움 (추가 증권사 확장 가능)
- 필드: 계좌번호, 계좌 별칭(선택), App Key, App Secret, 모의투자 토글
- 저장 성공 시 모달 닫기 + 목록 갱신
- 에러 시 모달 내 에러 메시지 표시

### FR-04 계좌 수정 모달
- FR-03과 같은 모달, 계좌번호는 disabled
- App Key / App Secret: 변경 시에만 입력 (placeholder: "••••••••")

### FR-05 계좌 활성/비활성 토글
- 행 우측 슬라이드 토글
- 비활성 계좌는 포트폴리오 집계에서 제외

### FR-06 계좌 검증
- 미검증 + API 연결됨 계좌에만 "검증" 버튼 표시
- 클릭 시 실제 API 호출로 연결 상태 확인
- 실패 시 alert: "계좌 검증 실패. API 키를 확인해 주세요."

### FR-07 계좌 삭제
- 행 우측 휴지통 아이콘
- 삭제 confirm 대화상자: "계좌를 삭제할까요? 이 작업은 되돌릴 수 없습니다."

### FR-08 순서 변경
- 행 우측 위/아래 화살표로 display_order 변경
- 첫 번째 계좌는 위 화살표 비활성, 마지막은 아래 화살표 비활성
- 계좌 1개이면 화살표 숨김

### FR-09 환경변수 계좌 (env-kis)
- `.env`의 KIS_APP_KEY / KIS_ACCOUNT_NO로 등록된 계좌는 id="env-kis"로 목록에 포함
- 이 계좌는 수정/삭제 불가 (UI에서 해당 버튼 비활성 또는 미표시 — 현재 미구현)
- 포트폴리오 홈 계좌 카드에도 표시되어야 함

### FR-10 지원 증권사 카드
- 등록 가능한 증권사 정보 카드 (브로커명, 코드, API 포털 링크)

### FR-11 API 키 발급 안내
- KIS / 키움 발급 방법 간략 안내 박스

## API 의존성

| API | 용도 |
|-----|------|
| `GET /api/v1/accounts` | 계좌 목록 조회 (env-kis synthetic 포함) |
| `POST /api/v1/accounts` | 계좌 추가 |
| `PATCH /api/v1/accounts/{id}` | 계좌 수정 |
| `DELETE /api/v1/accounts/{id}` | 계좌 삭제 |
| `POST /api/v1/accounts/{id}/verify` | 계좌 검증 |
| `PATCH /api/v1/accounts/{id}/toggle` | 활성/비활성 토글 |
| `POST /api/v1/accounts/{id}/reorder?direction=up\|down` | 순서 변경 |
| `GET /api/v1/accounts/brokers` | 지원 증권사 목록 |
