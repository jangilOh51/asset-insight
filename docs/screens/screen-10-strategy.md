# screen-10 투자 전략서 (`/strategy`)

> last_updated: 2026-04-20
> version: 1.0
> sprint: S3-02

---

## 화면 목적

리스크 성향과 투자 기간을 설정하면 Claude AI가 현재 포트폴리오 기반의
맞춤형 투자 전략 문서를 생성한다.

---

## UI 상태 (5가지)

| 상태 | 트리거 | 표시 |
|------|--------|------|
| 기본 | 페이지 최초 진입 | 리스크 성향 선택기 + 투자 기간 선택기 + "전략서 생성" 버튼 + 안내 |
| 로딩 | 버튼 클릭 후 | 스피너 + "AI가 전략서를 작성 중입니다..." |
| 성공 | API 응답 수신 | 전략서 7개 섹션 카드 |
| 에러 | API 오류 | 에러 메시지 + 재시도 버튼 |
| 빈 화면 | ANTHROPIC_API_KEY 미설정 | 설정 안내 |

---

## 레이아웃

```
AppLayout (title="투자 전략서")
└── max-w-3xl mx-auto
    ├── 설정 카드
    │   ├── 리스크 성향: 보수적 / 중립 / 공격적 (3-button toggle)
    │   ├── 투자 기간: 1년 / 3년 / 5년 / 10년 (4-button toggle)
    │   └── "전략서 생성" 버튼
    ├── 캐시 뱃지 (새로 생성됨 / 캐시됨)
    └── 전략서 영역 (7개 섹션 카드)
```
