# 시황 — 화면 요구사항

## 목적
주요 지수(KOSPI / S&P 500 / NASDAQ / USD/KRW)와 보유 종목 트리맵으로
현재 시장 상황을 한눈에 파악한다.

---

## 기능 요구사항

### FR-01 주요 지수 카드
- KOSPI / S&P 500 / NASDAQ / USD/KRW 4종 표시
- 각 카드: 지수명 / 현재가 / 등락폭(절댓값) / 등락률(%)
- 상승: #EF4444(빨강), 하락: #60A5FA(파랑), 보합: 회색
- 5분 간격 자동 갱신 (Redis TTL=300s)

### FR-02 보유 종목 트리맵
- 실시간 포트폴리오 보유 종목 기반 (SectorTreemap 컴포넌트 재사용)
- 셀 크기: 평가금액 비례, 셀 색상: 당일 등락률
- 보유 종목 없으면 빈 상태 표시

### FR-03 UI 5가지 상태
- 기본 / 로딩 / 에러 / 빈화면 / 오프라인

## 비기능 요구사항
