# screen-11 세금 시뮬레이터 (`/tax`)

> last_updated: 2026-04-20
> version: 1.0
> sprint: S3-03

---

## 화면 목적

보유 종목을 매도할 때 발생하는 예상 세금을 계산한다.
증권거래세, 양도소득세를 항목별로 표시하고 세후 순이익을 보여준다.

---

## UI 상태 (5가지)

| 상태 | 트리거 | 표시 |
|------|--------|------|
| 기본 | 페이지 진입 | 보유 종목 드롭다운 + 수량 입력 + "계산" 버튼 |
| 로딩 | 버튼 클릭 | 스피너 |
| 성공 | 계산 완료 | 세금 항목별 카드 + 세후 순이익 |
| 에러 | API 오류 | 에러 메시지 + 재시도 |
| 빈 화면 | 보유 종목 없음 | "보유 종목을 먼저 등록해주세요" |

---

## 레이아웃

```
AppLayout (title="세금 시뮬레이터")
└── max-w-2xl mx-auto
    ├── 입력 카드
    │   ├── 종목 선택 (보유 종목 드롭다운)
    │   ├── 매도 수량 입력 (숫자 input)
    │   └── "세금 계산" 버튼
    └── 결과 카드 (성공 시)
        ├── 매도 총액 / 매입 총액 / 양도차익
        ├── 세금 항목 (증권거래세 / 양도소득세)
        ├── 총 세금 합계
        ├── 세후 순이익 (강조)
        ├── 실효세율
        └── 주의사항 목록
```
