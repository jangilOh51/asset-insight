# 포트폴리오 상세 — 화면 요구사항

## 목적
단일 계좌의 실시간 잔고·종목·자산 구성을 상세 조회한다.
계좌 ID(`id`)를 URL 파라미터로 받아 해당 계좌 데이터만 표시한다.

---

## 기능 요구사항

### FR-01 계좌 요약 패널 (좌측/상단)
- 총 자산, 수익률, 매입금액, 평가금액, 평가손익, 예수금 StatRow로 나열
- 환율 표시 (해외 계좌 포함 시)
- 마지막 업데이트 시각 + 새로고침 버튼 (스피너 포함)
- 수익 색상: `#EF4444`, 손실: `#60A5FA`

### FR-02 자산 구성 도넛 차트
- 자산 유형별(국내주식 / 해외주식 / 현금) 비중을 도넛 SVG 차트로 표시
- 중앙에 가장 큰 비중 라벨 표시
- 우측에 범례 (유형명 + 비중%)
- SSR false (dynamic import 필수)

### FR-03 기간별 자산 변화 바차트
- DB 스냅샷 기반 월별 자산 변화 바차트 (최근 6개월)
- 최신 월 바는 `#06B6D4`, 과거는 `#1E3A5F`
- 데이터 없으면 "스냅샷 데이터 수집 중" 안내
- SSR false (dynamic import 필수)

### FR-04 종목 목록 (우측/하단)
- 전체 / 국내 / 해외 탭으로 필터링
- 정렬 기준: 평가금액 / 등락률 / 비중 토글 (등락률 = 당일 등락률 기준)
- 컬럼: 종목명, 시장 배지, 수량, 매입금액, 평가금액, 평가손익, **당일 등락률**
- 당일 등락률(`day_change_pct`): KIS API `prdy_ctrt` 필드 — 전일 종가 대비 오늘 변동률
- 누적 수익률(`return_pct`)은 종목 행에서 제거 (당일 등락률로 대체)
- 행 클릭 → 향후 종목 상세 (현재 미구현)
- 빈 상태: "보유 종목이 없습니다"

### FR-05 자동/수동 갱신
- 페이지 진입 시 즉시 로드
- 새로고침 버튼 클릭 시 silent=false로 재로드 (스피너 표시)
- 60초마다 silent 갱신

### FR-06 로딩 상태
- 좌측 패널: 스켈레톤 StatRow
- 도넛차트: 회색 원형 placeholder
- 바차트: 회색 바 placeholder
- 종목 테이블: 스켈레톤 행

### FR-07 에러 상태
- 빨간 배너 + "다시 시도" 버튼

### FR-08 네비게이션
- TopBar 좌측 "← 포트폴리오" 뒤로가기 링크
- TopBar 제목: 계좌번호 또는 계좌 별칭

## 비기능 요구사항

- 2열 레이아웃: 좌(360px 고정) + 우(flex-1)
- 모바일에서는 1열로 전환 (현재 미구현)
- 최대 너비: 1100px

---

## API 의존성

| API | 용도 |
|-----|------|
| `GET /api/v1/portfolio/realtime?account_id={id}` | 계좌 실시간 데이터 |
| `GET /api/v1/accounts` | 계좌 목록 (account_no 조회용) |
| `GET /api/v1/snapshot/summary/{account_no}?limit=180` | 월별 자산 변화 |
