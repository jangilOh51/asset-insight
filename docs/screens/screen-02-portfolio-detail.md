# 포트폴리오 상세 — 화면 요구사항

## 목적
단일 계좌의 실시간 잔고·종목·자산 구성을 상세 조회한다.
계좌 ID(`id`)를 URL 파라미터로 받아 해당 계좌 데이터만 표시한다.

---

## 기능 요구사항

### FR-01 계좌 요약 헤더 (상단 고정)
- 계좌명/브로커명 표시
- **요약 카드 5종** (그리드): 총 자산, 투자원금(매입금액), 평가손익(+수익률%), 예수금, 당일 등락
- 수익 색상: `#EF4444`, 손실: `#60A5FA`
- 마지막 업데이트 시각 + 새로고침 버튼 (스피너 포함)

### FR-02 자산 구성 차트 섹션
- **도넛 차트** (좌): 자산 유형별(국내주식 / 해외주식 / 현금) 비중
  - 중앙에 가장 큰 비중 라벨
  - 우측 범례 (유형명 + 비중%)
- **월별 자산 변화 바차트** (우): DB 스냅샷 기반 최근 6개월
  - 최신 월 바는 `#06B6D4`, 과거는 `#1E3A5F`
  - 데이터 없으면 "스냅샷 데이터 수집 중" 안내
- 두 차트는 좌우 나란히 (각 50% 너비)
- SSR false (dynamic import 필수)

### FR-02b 자산 배분 바 (차트 하단)
차트 섹션 바로 아래, 종목 테이블 위에 배치.

- **국내 / 해외 / 현금** 수평 분할 바
  - 국내: `#60A5FA`, 해외: `#F59E0B`, 현금: `#4B5563`
  - 우측에 각 비율 % 표시
- **ETF / 주식** 수평 분할 바 (현금 제외 순수 투자 금액 기준)
  - ETF: `#34D399`, 주식: `#A78BFA`
  - 우측에 각 비율 % 표시
- ETF 판별 규칙 (KR): 이름 prefix — TIGER, KODEX, ARIRANG, KBSTAR, ACE, HANARO, SOL, KOSEF 또는 이름에 "ETF" 포함
- ETF 판별 규칙 (US): 이름에 "ETF" 포함

### FR-03 종목 테이블 (풀컬럼)
therich.io 스타일의 전체 컬럼 표시 테이블

#### 탭 필터
- **전체 / 국내 / 해외 / 자산유형** 탭
- 국내·해외 탭은 해당 종목 수 배지 표시

#### 자산유형 탭 — 그룹 뷰
- 자산 그룹: **국내 주식 / 국내 ETF / 해외 주식 / 해외 ETF**
- 그룹 색상: 국내주식 `#60A5FA`, 국내ETF `#34D399`, 해외주식 `#F59E0B`, 해외ETF `#A78BFA`
- 그룹 헤더 행: 좌측 컬러 바 + 그룹명 + 종목 수 + 접기/펼치기 토글
- 그룹 헤더 소계: 매입금액, 평가금액, 평가손익, 수익률, 비중%
- 그룹 내 종목: 평가금액 내림차순 고정 정렬
- 그룹 순서: 평가금액 합계 내림차순

#### 전체/국내/해외 탭 — 기존 뷰
컬럼 헤더 클릭 정렬 (오름/내림 토글), 기본 정렬: 평가금액 내림차순

#### 컬럼 (좌 → 우)
| 컬럼 | 설명 |
|------|------|
| 종목명 | 이름 + 심볼 + 시장 배지 |
| 현재가 | 현재 가격 (네이티브 통화) |
| 수량 | 보유 수량 |
| 평균단가 | 평균 매입 단가 (네이티브 통화) |
| 매입금액 | 총 매입금액 (KRW) |
| 평가금액 | 총 평가금액 (KRW) |
| 평가손익 | 손익 금액 (KRW) |
| 수익률 | 누적 수익률 (%) |
| 당일 등락 | 당일 등락률 (`day_change_pct`) |
| 비중 | 포트폴리오 내 비중 (%) |
| AI | AI 종목 분석 버튼 |

#### 당일 등락 세부
- KIS API `prdy_ctrt` 필드 (전일 종가 대비)
- `prdy_ctrt` == 0: DB 스냅샷 fallback → `전일` 배지 표시

### FR-04 자동/수동 갱신
- 페이지 진입 시 즉시 로드
- 새로고침 버튼 클릭 시 스피너 표시
- 60초마다 silent 갱신

### FR-05 로딩 상태
- 요약 카드: 스켈레톤 펄스
- 차트 영역: 회색 placeholder
- 종목 테이블: 스켈레톤 행

### FR-06 에러 상태
- 빨간 배너 + "다시 시도" 버튼

### FR-07 네비게이션
- TopBar 좌측 "← 포트폴리오" 뒤로가기 링크
- TopBar 중앙: 계좌번호 또는 계좌 별칭

## API 의존성

| API | 용도 |
|-----|------|
| `GET /api/v1/portfolio/realtime?account_id={id}` | 계좌 실시간 데이터 |
| `GET /api/v1/accounts` | 계좌 목록 (account_no 조회용) |
| `GET /api/v1/snapshot/summary/{account_no}?limit=180` | 월별 자산 변화 |
