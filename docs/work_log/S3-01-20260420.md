## [백엔드] 구현

### 작업 내용
1. `backend/pyproject.toml` — `anthropic>=0.40.0` 의존성 추가
2. `backend/app/core/config.py` — `anthropic_api_key: str = ""` 필드 추가
3. `.env.example` — `ANTHROPIC_API_KEY=` 항목 추가
4. `backend/app/services/analysis/report.py` — Claude API 서비스 신규 생성
   - `claude-opus-4-7` 모델 사용
   - `thinking: {"type": "adaptive"}` 적용
   - 시스템 프롬프트에 `cache_control: {"type": "ephemeral"}` 적용 (프롬프트 캐싱)
   - 리포트 5섹션 한국어 생성
5. `backend/app/api/v1/endpoints/report.py` — `POST /api/v1/report/monthly` 신규 생성
   - Redis 캐시 확인 (키: `report:monthly:{year}:{month}`, TTL: 3600s)
   - `UnifiedSummary` → `HoldingItem` + `PortfolioSummary` 변환 함수 분리
   - `ValueError` → 503, 기타 예외 → 502 처리
6. `backend/app/api/v1/router.py` — `report_router` 등록
