> 상태: ⬜ 대기

> 작성일: 2026-04-19

---

## 목적 / 배경

백엔드 트렌드 API(`/api/v1/trend/{account_id}`)는 구현되어 있으나 프론트엔드 화면이 없다.  
일/주/월/연 단위 자산 변화 추이를 차트로 시각화하는 `/trend` 페이지를 구현한다.

## 작업 상세

### [백엔드] 트렌드 API 검증
- `GET /api/v1/trend/{account_id}?period=daily&limit=90` 응답 확인
- 응답 필드: `time`, `total_value_krw` 포함 여부
- 벤치마크 데이터(KOSPI/S&P500) 별도 API 또는 정적 데이터 방식 결정

### [프론트] 파일 생성
```
frontend/pages/trend/index.tsx          ← 트렌드 메인 페이지
frontend/components/charts/AssetTrendChart.tsx  ← 라인차트
frontend/components/charts/MonthlyBarChart.tsx  ← 월별 막대차트
```

### [프론트] 차트 구현 방식
- **라인차트**: 순수 SVG path (recharts 미사용, SSR safe)
- **막대차트**: flex 기반 CSS (AssetBarHistory 패턴 참고)
- 축 레이블: JetBrains Mono
