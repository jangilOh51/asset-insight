## 목적 / 배경

트렌드 화면에 KOSPI / S&P500 / NASDAQ 수익률을 내 포트폴리오와 오버레이로 비교한다.  
"내 투자 성과가 시장 대비 얼마나 좋은가"를 시각적으로 확인할 수 있게 한다.

| 프론트 개발자 | Claude/프론트 | 멀티라인 오버레이 차트 구현 |

---

## 인수 기준 (Acceptance Criteria)

- [ ] 트렌드 차트에 내 포트폴리오 + KOSPI + S&P500 라인이 동시에 표시된다
- [ ] 각 라인은 색상으로 구분되고 범례가 있다
- [ ] 시작일 기준 수익률(%)로 정규화하여 비교한다
- [ ] 벤치마크 데이터 미수집 시 점선으로 "데이터 없음" 표시

---

## 데이터 전략

### 벤치마크 데이터 수집 방법 (우선순위 순)
1. **Yahoo Finance API** (무료, 비공식) — `^KS11` (KOSPI), `^GSPC` (S&P500), `^IXIC` (NASDAQ)
2. **한국거래소 Open API** — KOSPI 공식 데이터
3. **정적 CSV 파일** — 개발 초기 임시 데이터

### 정규화 방식
```
기준일 수익률 = (현재값 / 시작값 - 1) × 100
내 포트폴리오: 첫 스냅샷 기준
KOSPI: 첫 스냅샷과 동일 날짜 기준
```
